## 🛠 Development (개발)
*“남들이 해본 것부터 아무도 가보지 않은 길까지, 가리지 않고 구현합니다.”*

- **Mainstream:** 남들 다 해본 대중적인 기술 스택
- **Niche:** 남들이 시도하지 않은 독특한 아이디어
- **Fundamentals:** 탄탄한 기초 다지기
- **Challenge:** 한계를 시험하는 고난도 아키텍처 설계
- **Mindset:** **"그냥 다 해본다"**

## 🧪 Testing (테스트)
*“직접 만든 코드부터 세상의 모든 서비스까지, 철저하게 분석합니다.”*

- **Self-Build:** 내가 직접 개발한 프로덕트 검증
- **Third-Party:** 남이 만든 서비스 분석 및 벤치마킹
- **Interests:** 개인적으로 호기심이 생기는 모든 것
- **Trends:** 지금 이 순간 유행하는 최신 기술
- **Mindset:** **"그냥 다 해본다"**

## 💰 Service (서비스)
